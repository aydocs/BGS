## Temiz Kod Nedir?

Temiz kod, kolay okunabilir, anlaşılabilir ve bakım yapılabilir kod demektir. Robert C. Martin'in “Clean Code” kitabı bu konudaki en önemli kaynaklardan biridir.

---

## Temiz Kodun Temel Prensipleri

- **Anlamlı İsimlendirme:** Fonksiyon, değişken, sınıf isimleri amaçlarını açıkça belirtmeli.
- **Fonksiyonlar Küçük Olmalı:** Tek bir iş yapmalı.
- **Yinelenen Koddan Kaçının:** DRY prensibi.
- **Yorumlar:** Gerektiğinde açıklayıcı ve kısa olmalı.
- **Kod Düzeni:** Tutarlı boşluklar ve girintileme.

---

## Örnek: Temiz ve Kirli Kod Karşılaştırması

```javascript
// Kirli kod
function d(a,b){return a+b;}

// Temiz kod
function topla(sayi1, sayi2) {
  return sayi1 + sayi2;
}
