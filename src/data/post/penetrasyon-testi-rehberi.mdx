## Penetrasyon Testi Araçları

```bash
# Nmap ile Port Tarama
nmap -sS -sV -O target.com

# Nikto ile Web Zafiyet Tarama
nikto -h http://target.com

# Sqlmap ile SQL Injection Testi
sqlmap -u "http://target.com/page?id=1" --dbs
```
